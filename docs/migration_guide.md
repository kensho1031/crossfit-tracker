# 🚀 プロジェクトお引っ越し＆同期ガイド

このガイドでは、今のパソコン（PC A）から新しいパソコン（PC B）へ引っ越しをして、また戻ってくる方法を「冒険」に例えて説明します！

---

## 🎒 パソコン A (今のパソコン)：冒険の記録を預けよう

1.  **カバンを整理する**:
    冒険の記録（`docs` フォルダ）を、プログラムが入っているカバン（`crossfit-tracker` フォルダ）の中に入れました。これで準備OK！

2.  **「物置き」に預ける**:
    GitHubという「世界中のどこからでも開ける魔法の物置き」に、カバンを預けました。
    *   **実行した呪文**: `git add .` -> `git commit` -> `git push`

---

## 🏗️ パソコン B (新しいパソコン)：冒険を再開しよう

1.  **カバンをコピーする (git clone)**:
    魔法の物置きから、カバンをコピーして自分のパソコンに持ってきます。
    *   **呪文**: `git clone https://github.com/kensho1031/crossfit-tracker.git`

2.  **部品を組み立てる (npm install)**:
    プログラムが動くように、必要な部品をインターネットから自動で集めます。
    *   **呪文**: `npm install`

3.  **内緒の合言葉を入れる (.env)**:
    パスワードのような「内緒の合言葉」は物置きには入っていません。前のパソコンからコピーして、自分だけの `.env` ファイルを作って貼り付けます。

4.  **魔法使い（Antigravity）に頼む**:
    Antigravityを起動して、「カバンの中にある `docs` を読んで、続きをして！」と言います。これで魔法使いは、これまでの冒険を思い出してくれます！

5.  **終わったらまた預ける**:
    作業が終わったら、また物置き（GitHub）にカバンを戻します。

---

## ✨ パソコン Aに戻ってきたら：最新の状態にする

別のパソコンで進めた分を、今のパソコンに取り込みます。

*   **呪文 (git pull)**:
    物置きから「新しい分だけ」をカバンに入れます。
    *   **呪文**: `git pull origin main`

---

### 💡 魔法使い（Antigravity）と仲良くするコツ
*   **タスクリスト (`task.md`)**: 次に何をやるか、魔法使いと共有するメモです。
*   **歩んだ道 (`walkthrough.md`)**: 今日何をしたか、魔法使いが日記をつけてくれます。
*   **計画書 (`implementation_plan.md`)**: これからやる大きな工事の設計図です。

これらが `docs` フォルダに入っていれば、別のパソコンに行っても魔法使いはすべてを覚えています！
