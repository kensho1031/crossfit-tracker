# CrossFit Tracker ベータテストガイド

この度はベータテストにご協力いただきありがとうございます！
スマホで「アプリ」として快適にご利用いただくための手順をまとめました。

### 1. スマホからアクセスする
1. PCとスマホを**同じWi-Fi**に接続します。
2. スマホのブラウザ（SafariやChrome）で以下のURLを入力します：
   `http://192.168.68.110:5173`
   （※PCのIPアドレスが変わった場合は、PCのターミナルで `ipconfig` を実行して確認してください）

> [!TIP]
> PCのIPアドレスは、コマンドプロンプトやターミナルで `ipconfig` (Windows) または `ifconfig` (Mac) と打つことで確認できます。

## 2. ホーム画面に追加する（PWA機能）

PWA（Progressive Web App）に対応しました。ホーム画面に追加することで、ブラウザの枠がなくなり、フルスクリーンの「アプリ」として動作します。

### iPhone (Safari):
1. 下部の「共有ボタン」（上矢印のアイコン）をタップ。
2. 「ホーム画面に追加」を選択。

### Android (Chrome):
1. 右上の「3点リーダー」メニューをタップ。
2. 「アプリをインストール」または「ホーム画面に追加」を選択。

## 3. テストしてほしいポイント
- **Big 3 の視認性**: ゲージがスマホ画面で見やすく、値が正しく反映されているか。
- **WODスキャン**: スマホのカメラでボードを撮影し、自動分類がうまく機能するか。
- **編集のしやすさ**: 種目の修正やセクションの移動が指でスムーズに行えるか。

## 4. フィードバックについて
問題が発生したり、改善案がある場合はいつでもお知らせください。
特に「表示が崩れている箇所」などがあれば、画面サイズを添えて教えていただけると助かります。
